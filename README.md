news
===========

Usage
-----

1. 安裝依賴

        $ npm install -g gulp
        $ npm install
        $ bower install

1. 配置 gulp config

        $ cp _config.coffee config.coffee
        $ vi config.coffee

1. 配置 client config

        $ cp app/config/_config.coffee app/config/config.coffee
        $ vi app/config/config.coffee

1. 啟動應用

        $ gulp

1. 打包

        $ gulp build -P
